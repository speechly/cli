## speechly transcribe

Transcribe the given jsonlines file

```
speechly transcribe <input_file> [flags]
```

### Examples

```
speechly transcribe <input_file>
```

### Options

```
  -h, --help           help for transcribe
  -m, --model string   On-device model bundle file
```
